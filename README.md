# 📖 Project Overview

This project presents an **Exploratory Data Analysis (EDA)** of customer behavior for **ConnectaTel**, a telecommunications company operating in Latin America.

Using historical customer data through **2024**, the analysis explores service usage patterns, customer segmentation, and churn behavior to generate actionable business insights that can support decision-making and improve customer retention strategies.

The project follows a complete data analysis workflow including data cleaning, validation, descriptive statistics, exploratory analysis, visualization, and business recommendations.

---

# 🎯 Business Objectives

The main objectives of this analysis are to:

- Analyze customer usage behavior.
- Identify consumption patterns across different service plans.
- Detect abnormal or atypical customer behavior.
- Understand customer churn.
- Compare customer segments.
- Generate business recommendations based on data.

---

# 📂 Dataset Description

The project uses three datasets.

| Dataset | Description |
|---------|-------------|
| 📄 **plans.csv** | Information about available service plans, monthly prices, included minutes, included data, and additional charges. |
| 👤 **users_latam.csv** | Customer demographic information including age, city, registration date, subscribed plan, and churn status. |
| 📞 **usage.csv** | Customer service usage including call minutes and text messages. |

# ▶️ How to Run the Project

### 1️⃣ Clone the repository


### 3️⃣ Install dependencies


### 4️⃣ Open the notebook

Launch Jupyter Notebook and open:


Run all notebook cells to reproduce the complete analysis.
